Поддержка стилей и скриптов bootstrap для всех элементов CF7. Алерты, спиннеры, валидация.

== Описание ==

Интеграция функционала bootstrap с плагином "Contact Form 7".

== FAQ ==

= Поддерживаются ли алерты? =

Да. После активации плагина, все сообщения будут помещены внутрь элемента <code>.alert</code>.
